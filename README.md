# Iron Barbell Club

Projet HTML/CSS/JS statique pour le site Iron Barbell Club.

## Structure

```text
index.html
assets/
  css/
    variables.css
    style.css
    responsive.css
  js/
    main.js
  images/
    logo/
    disciplines/
favicon.ico
robots.txt
sitemap.xml
```

## Mise en ligne sur GitHub

1. Copier tous les fichiers dans ton dépôt GitHub.
2. Commit/push :

```bash
git add .
git commit -m "Refactorisation du site Iron Barbell Club"
git push
```

## À modifier avant publication

Dans `robots.txt` et `sitemap.xml`, remplace `https://example.com` par l'adresse réelle du site.
